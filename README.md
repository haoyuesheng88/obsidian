# Obsidian Skills

This repository stores reusable Codex skills related to local Obsidian workflows.

## Included skills

- `skills/write-to-obsidian`: create, update, or append Markdown notes in the active local Obsidian vault on Windows.

## Use on another computer

1. Clone this repository.
2. Copy the skill folder into your Codex skills directory, usually `~/.codex/skills/`.
3. Use `$write-to-obsidian` in a prompt.
